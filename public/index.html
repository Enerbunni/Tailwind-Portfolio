Islands of Asrora - Game Jam Submission

Islands of Asrora is a platformer adventure game I created in 2 weeks for Adventure Jam 2022. The goal of the player is to transport an evil gem to a far-off island to be destroyed by manipulating the objects around them to create platforms. I made the game in Unity and it was my first time fully going through the game development cycle of a game in 3D space.

I wanted to make the game centered around something interesting so I came up with a mechanic where you can 'posses' the objects on the various islands. This allows you to move and rotate them however you want in the world. You can then either freeze or release those objects, returning control and camera to the player, effectively letting the player build their own way to each island.
